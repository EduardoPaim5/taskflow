# Nexilum Frontend

Frontend React + TypeScript do Nexilum.

## Requisitos
- Node.js 20 LTS ou superior
- Backend Nexilum em execução

## Configuração
1. Instale dependências:
```bash
npm install
```

2. Configure a URL da API em `.env`:
```bash
VITE_API_BASE_URL=http://localhost:8080/api
```

3. Rode em modo desenvolvimento:
```bash
npm run dev
```

## Scripts
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
